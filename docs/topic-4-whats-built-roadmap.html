Roadmap
Known gaps in the prototype — addressed in the roadmap
⚠

No API authentication — endpoints are open. MVP adds API key auth with per-product scoping.

⚠

Transform execution incomplete — field copies work, but mathematical transforms (multiply, round, date) are not yet wired at runtime.

⚠

No rate limiting or payload size guards — production hardening needed before real traffic.

⚠

No configuration audit trail — who changed what, when. Required for compliance-ready deployments.

⚠

No auto-scaling — static task count on ECS. Needs target-tracking scaling for core-rating and rules-service.

⚠

Redis running but not integrated — flow definitions, mappings, and rules hit the database on every request.
